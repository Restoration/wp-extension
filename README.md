##概要
WordPress案件様に使用しているプラグイン
functions.phpに記述するのではなく
自作プラグインを開発し、それらを拡張して使用するようにしている。

##主要機能一覧
- カスタムポストタイプ
- 管理画面ログイン時のロゴ変更
- WordPressバージョンと更新を非表示にする
- ダッシュボードの不要なウィジェットを消す
- 投稿画面の項目をカスタマイズ
- 管理画面内のメニューを非表示にする
- 管理バーの項目をカスタマイズ
- 記事公開時にアラートを出す
- ログアウト後のページ遷移変更
- 404へリダイレクト
- ダッシュボードにウィジェットを追加
- メディアライブラリにカテゴリー、タグ機能を追加
